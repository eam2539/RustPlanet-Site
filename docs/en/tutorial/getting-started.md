# Quick Start

Welcome to RustPlanet! This guide will help you get up and running quickly.

## System Requirements

RustPlanet supports the following platforms:

- Windows 7 or later
- Linux (with OpenGL support)
- macOS 10.13 or later
- Android 6.0 or later

## Installation

### Windows / macOS / Linux

1. Download RustPlanet from Steam
2. Run the main executable:
   - Windows: `RustPlanet.exe`
   - macOS: `RustPlanet.app`
   - Linux: `RustPlanet`

### Android

RustPlanet is free on Android with no additional cost.

1. Download RustPlanet from Google Play
2. Install the APK on your Android device
3. Launch RustPlanet from your app list

## Basic Controls

### Game Controls

- **Move Camera**: Use W/A/S/D keys or arrow keys
- **Rotate View**: Hold right mouse button and move mouse
- **Zoom**: Mouse wheel or two-finger pinch on touch screen
- **Select Units**: Left-click or drag a selection box
- **Issue Commands**: Right-click on target location

### Unit Management

- **Build Units**: Select a building and click the unit icon to build
- **Form Squads**: Hold Ctrl + number key to assign a squad
- **Quick Select Squad**: Press the corresponding number key

## FAQ

### Q: Game won't start?
A: Please ensure:
1. Your system meets the minimum requirements
2. Graphics drivers are up to date

### Q: How to report issues?
A: Please submit issues on the [GitHub Issues](https://github.com/eam2539/RustPlanet-Site/issues) page with:
1. Detailed description of the issue
2. System environment information
3. Error logs (if any)
4. Steps to reproduce
