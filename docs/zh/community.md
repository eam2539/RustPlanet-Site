## 参与贡献

### 提交问题

发现 bug 或有新功能建议？请在 GitHub 上提交 issue：
1. 访问 [GitHub Issues](https://github.com/eam2539/RustPlanet/issues)
2. 点击 "New Issue"
3. 选择适当的模板
4. 详细描述你的问题或建议

## 联系我们

- GitHub：[eam2539](https://github.com/eam2539)

## 常见问题

### Q: 在哪里可以找到其他开发者？
A: 加入我们的 Discord 服务器或 QQ 群，那里有活跃的开发者社区。

### Q: 如何报告 bug？
A: 在 GitHub 上提交 issue，使用 bug 报告模板，并提供尽可能详细的信息。

## 支持项目

如果你喜欢 RustPlanet 项目，可以通过以下方式支持我们：

1. Star 我们的 GitHub 仓库
2. 分享项目给其他人
3. 提供反馈和建议
4. 帮助回答社区问题

你的每一份支持都对项目的发展至关重要！
