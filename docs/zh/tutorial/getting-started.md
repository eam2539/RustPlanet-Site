# 快速开始

欢迎来到 RustPlanet！本指南将帮助你快速上手并开始使用 RustPlanet。

## 系统要求

RustPlanet 支持以下平台：

- Windows 7 或更高版本
- Linux (带有 OpenGL 支持)
- macOS 10.13 或更高版本
- Android 6.0 或更高版本

## 安装

### Windows / macOS / Linux

1. 从Steam下载RustPlanet
2. 运行主程序文件：
   - Windows: `RustPlanet.exe`
   - macOS: `RustPlanet.app`
   - Linux: `RustPlanet`

### Android

RustPlanet 在Android平台免费提供，无需任何费用。

1. 从 Google Play 下载 RustPlanet
2. 在 Android 设备上安装 APK
3. 从应用列表启动 RustPlanet

## 基本操作

### 游戏控制

- **移动摄像机**：使用 W/A/S/D 键或方向键
- **旋转视角**：按住右键并移动鼠标
- **缩放**：鼠标滚轮或触屏双指缩放
- **选择单位**：左键点击或框选
- **发布命令**：右键点击目标位置

### 单位管理

- **建造单位**：选择建筑物，点击要建造的单位图标
- **编队**：按住 Ctrl + 数字键设置编队
- **快速选择编队**：按对应数字键

## 常见问题

### Q: 游戏无法启动？
A: 请确保：
1. 系统满足最低要求
2. 显卡驱动已更新

### Q: 如何报告问题？
A: 请在 [GitHub Issues](https://github.com/eam2539/RustPlanet-Site/issues) 页面提交问题，并提供：
1. 问题详细描述
2. 系统环境信息
3. 错误日志（如果有）
4. 重现步骤
